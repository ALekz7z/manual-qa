## 1. Введение

### 1.1 Цель документа
Этот тест-план описывает стратегию ручного тестирования веб-сайта SauceDemo.

### 1.2 Описание проекта
SauceDemo — учебный интернет-магазин для тестирования, имитирующий процесс покупки товаров.

**URL:** https://www.saucedemo.com/  
**Тип приложения:** Веб-сайт (E-Commerce)

---

## 2. Область тестирования

### Что тестируем:
- Авторизация пользователей (логин/пароль)
- Просмотр каталога товаров
- Добавление товаров в корзину
- Оформление заказа (Checkout)
- Базовый UI/UX

### Что НЕ тестируем:
- Платёжные системы
- Нагрузочное тестирование
- Безопасность и penetration testing
- Мобильные приложения
- API тестирование
- Автоматизированные тесты
