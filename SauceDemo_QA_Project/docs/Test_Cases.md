## TC-001: Успешная авторизация стандартного пользователя

| Параметр | Значение |
|----------|----------|
| **ID** | TC-001 |
| **Название** | Успешная авторизация стандартного пользователя |
| **Приоритет** | High |
| **Тип** | Позитивный |
| **Предусловие** | Пользователь находится на странице https://www.saucedemo.com/ |
| **Тестовые данные** | Username: `standard_user`, Password: `secret_sauce` |

### Шаги выполнения:

| № | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Открыть браузер и перейти на https://www.saucedemo.com/ | Страница логина загрузилась |
| 2 | В поле "Username" ввести `standard_user` | Текст отображается в поле |
| 3 | В поле "Password" ввести `secret_sauce` | Символы скрыты (точки) |
| 4 | Нажать кнопку "Login" | Происходит переход на страницу товаров |

**Ожидаемый результат:**  
 Пользователь успешно авторизован  
 Открыта страница каталога товаров (`/inventory.html`)  
 Отображается заголовок "Products"  
 Видны товары для покупки

**Фактический результат:**  Соответствует ожидаемому  
**Статус:**  Passed  
**Скриншот:** screenshots/TC-001_login_success.png

 Кнопка сменилась на "Remove"  
 В значке корзины появилась цифра "1"  
 Товар отображается в корзине  
 Цена товара указана верно ($29.99)

## TC-003: Оформление заказа с валидными данными

| Параметр | Значение |
|----------|----------|
| **ID** | TC-003 |
| **Название** | Оформление заказа с валидными данными |
| **Приоритет** | Critical |
| **Тип** | Позитивный |
| **Предусловие** | Товар "Sauce Labs Backpack" находится в корзине |
| **Тестовые данные** | First Name: `John`, Last Name: `Doe`, Zip: `12345` |

### Шаги выполнения:

| № | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Находясь в корзине, нажать кнопку "Checkout" | Открывается форма оформления заказа |
| 2 | В поле "First Name" ввести `John` | Текст отображается в поле |
| 3 | В поле "Last Name" ввести `Doe` | Текст отображается в поле |
| 4 | В поле "Zip/Postal Code" ввести `12345` | Текст отображается в поле |
| 5 | Нажать кнопку "Continue" | Переход на страницу подтверждения |
| 6 | Проверить данные заказа и нажать "Finish" | Заказ оформлен |

**Ожидаемый результат:**  
 Форма приняла валидные данные  
 Произошёл переход на страницу подтверждения  
 После нажатия "Finish" заказ оформлен  
 Отображается сообщение "Thank you for your order!"  
 Присвоен номер заказа  
 Корзина пуста (цифра исчезла)

**Фактический результат:**  Соответствует ожидаемому  
**Статус:**  Passed  
**Скриншот:** screenshots/TC-003_order_complete.png
