## TC-001: Успешный вход в систему с корректными учётными данными

| Параметр | Значение |
|----------|----------|
| **Модуль** | Login |
| **Приоритет** | High |
| **Тип** | Positive |
| **Предусловие** | Пользователь находится на странице логина |

**Шаги:**
1. Перейти на https://opensource-demo.orangehrmlive.com/
2. В поле Username ввести: `Admin`
3. В поле Password ввести: `admin123`
4. Нажать кнопку "Login"

**Ожидаемый результат:**
- Пользователь авторизован
- Открыта страница Dashboard
- Отображается имя пользователя в правом верхнем углу
- Доступны все меню модулей (PIM, Leave, Time и т.д.)

## TC-004: Поиск сотрудника по имени

| Параметр | Значение |
|----------|----------|
| **Модуль** | PIM → Employee List |
| **Приоритет** | High |
| **Тип** | Positive |
| **Предусловие** | Пользователь авторизован, находится на странице Employee List |

**Шаги:**
1. Перейти в меню PIM → Employee List
2. В поле "Employee Name" ввести: `John`
3. Нажать кнопку "Search"

**Ожидаемый результат:**
- Отображаются только сотрудники с именем John
- Если сотрудников нет — сообщение "No Records Found"
- Фильтр применён корректно
