# OrangeHRM QA Project — Тестирование HR-системы

## Описание проекта

**Цель:** Провести ручное функциональное тестирование демо-версии HR-системы OrangeHRM.

**Объект тестирования:** https://opensource-demo.orangehrmlive.com/  
**Логин для доступа:** Admin / admin123

**Период тестирования:** 26.03.2026 — 27.03.2026  
**Тестировщик:** Бухтияров Алексей Андреевич

---

## Модули для тестирования

| Модуль | Страница | Что проверять |
|--------|----------|---------------|
| **Login** | /auth/login | Вход с правильными/неправильными данными, блокировка после неудачных попыток, восстановление пароля |
| **PIM → Employee List** | /pim/viewEmployeeList | Поиск сотрудника, фильтрация по статусу, пагинация, сортировка колонок |
| **Leave → Apply Leave** | /leave/viewLeaveList | Валидация дат (нельзя выбрать прошлое), обязательные поля, типы отпусков |
| **Dashboard** | /dashboard | Отображение виджетов, актуальность данных, рабочие ссылки на модули |
