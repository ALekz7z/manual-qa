## TC-001: Валидация формы Practice Form

| Параметр | Значение |
|----------|----------|
| **ID** | TC-001 |
| **Название** | Валидация формы Practice Form |
| **Приоритет** | Critical |
| **Тип** | Позитивный и Негативный |
| **Предусловие** | Пользователь находится на странице https://demoqa.com/practice-form |
| **Тестовые данные** | См. шаги выполнения |

### Шаги выполнения:

| № | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Открыть браузер и перейти на https://demoqa.com/practice-form | Страница формы загрузилась, все поля отображаются |
| 2 | Оставить поле "First Name" пустым, заполнить остальные поля, нажать Submit | Появляется сообщение об ошибке у поля First Name |
| 3 | Заполнить First Name: `John`, Last Name: `Doe` | Текст отображается в полях |
| 4 | В поле Email ввести невалидное значение: `invalidemail` | Появляется сообщение об ошибке валидации email |
| 5 | В поле Email ввести валидное значение: `test@example.com` | Текст принимается без ошибки |
| 6 | В поле Mobile ввести невалидное значение: `abc123` | Появляется сообщение об ошибке валидации телефона |
| 7 | В поле Mobile ввести валидное значение: `+1234567890` | Текст принимается без ошибки |
| 8 | Заполнить все обязательные поля корректными данными, нажать Submit | Форма успешно отправлена, появляется подтверждение |

**Ожидаемый результат:**  
- Поля с валидацией показывают ошибки при некорректном вводе  
- Форма отправляется только при корректном заполнении всех обязательных полей  
- После успешной отправки появляется модальное окно с подтверждением

**Фактический результат:** Соответствует ожидаемому  
**Статус:**  Пройденный  
**Скриншот:** screenshots/TC-001_practice_form_validation.png

## TC-003: Тестирование браузерных алертов

| Параметр | Значение |
|----------|----------|
| **ID** | TC-003 |
| **Название** | Тестирование браузерных алертов |
| **Приоритет** | High |
| **Тип** | Позитивный |
| **Предусловие** | Пользователь находится на странице https://demoqa.com/alerts (вкладка Browser Alerts) |
| **Тестовые данные** | Не требуются |

### Шаги выполнения:

| № | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Открыть браузер и перейти на https://demoqa.com/alerts | Страница алертов загрузилась, видна вкладка Browser Alerts |
| 2 | Переключиться на вкладку "Browser Alerts" (если не активна) | Вкладка активна, отображаются кнопки для работы с алертами |
| 3 | Нажать кнопку "Alert box" | Появляется системный алерт с сообщением |
| 4 | Нажать OK в алерте | Алерт закрывается, появляется подтверждение действия |
| 5 | Нажать кнопку "Confirm alert box" | Появляется алерт с кнопками OK и Cancel |
| 6 | Нажать OK в(confirm) алерте | Алерт закрывается, фиксируется выбор OK |
| 7 | Нажать кнопку "Confirm alert box" снова | Появляется алерт с кнопками OK и Cancel |
| 8 | Нажать Cancel в(confirm) алерте | Алерт закрывается, фиксируется выбор Cancel |
| 9 | Нажать кнопку "Prompt box" | Появляется алерт с полем ввода |
| 10 | Ввести текст в поле ввода и нажать OK | Текст принимается, появляется подтверждение |

**Ожидаемый результат:**  
- Alert box показывает сообщение и кнопку OK  
- Confirm alert box показывает кнопки OK и Cancel  
- Prompt box показывает поле для ввода текста  
- Все алерты корректно обрабатывают выбор пользователя

**Фактический результат:** Соответствует ожидаемому  
**Статус:** Пройденный

## TC-004: Перетаскивание элемента (Draggable)

| Параметр | Значение |
|----------|----------|
| **ID** | TC-004 |
| **Название** | Перетаскивание элемента (Draggable) |
| **Приоритет** | Medium |
| **Тип** | Позитивный |
| **Предусловие** | Пользователь находится на странице https://demoqa.com/draggable |
| **Тестовые данные** | Не требуются |

### Шаги выполнения:

| № | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Открыть браузер и перейти на https://demoqa.com/draggable | Страница загрузилась, отображается перетаскиваемый элемент |
| 2 | Найти элемент с надписью "Drag me" | Элемент виден, имеет отличный визуальный стиль |
| 3 | Зажать левую кнопку мыши на элементе "Drag me" | Курсор меняется, показывая возможность перетаскивания |
| 4 | Перетащить элемент в другое место страницы | Элемент перемещается вместе с курсором |
| 5 | Отпустить кнопку мыши | Элемент остаётся в новой позиции |
| 6 | Проверить координаты элемента (если отображаются) | Координаты изменились относительно начальной позиции |
| 7 | Перетащить элемент обратно в исходную позицию | Элемент возвращается на место |

**Ожидаемый результат:**  
- Элемент "Drag me" можно захватить мышью  
- При перетаскивании элемент следует за курсором  
- После отпускания элемент остаётся в новой позиции  
- Координаты позиции обновляются (если есть индикатор)

**Фактический результат:** Соответствует ожидаемому
